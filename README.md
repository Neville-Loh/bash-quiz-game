
# Jeopardy Quiz Game
This is a text based game developed using Bash based on the famous American TV shows, Jeopardy.


# Getting Started
```
> chmod +x jeopardy.sh
> ./jeopardy.sh
```

# Requirement 
Linux base enviroment
Text to speech module Festival in enviroment variable.
